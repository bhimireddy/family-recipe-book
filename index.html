spreadsheet setup

how to connect your spreadsheet:
  1. Upload your family-recipe-book.xlsx to your GitHub repository
  2. In GitHub, click the file → click Raw button → copy that URL
  3. Paste the raw URL below and click Save
  4. Click Refresh to load your recipes
Raw URL format looks like:
https://raw.githubusercontent.com/USERNAME/REPO/main/family-recipe-book.xlsx

updating your recipes

To add or edit recipes:
1. Open family-recipe-book.xlsx on your Mac
2. Add a new column for each new recipe
3. Go to your GitHub repository
4. Click Add file → Upload files
5. Upload the updated spreadsheet (same filename)
6. Click Commit changes
7. Come back here and click Refresh
